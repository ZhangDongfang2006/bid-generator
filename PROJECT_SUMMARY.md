## 🚀 立即启动

### 方法1：快速启动（推荐）

```bash
cd ~/workspace/bid-generator
./setup.sh
```

### 方法2：手动启动

```bash
cd ~/workspace/bid-generator
streamlit run app.py
```

浏览器将自动打开：http://localhost:8501

## 🎯 核心功能

### 1. 生成投标文件
- 上传招标文件（PDF/Word）
- 自动解析招标要求
- 智能匹配公司资料
- 生成投标文件（Word）

### 2. 资料管理
- 资质管理
- 案例管理
- 产品管理
- 人员管理

### 3. 系统设置
- 使用说明
- 公司信息
- 数据重置

## 📊 已预置数据

- 资质：2 条
- 案例：2 条（中天钢铁、汉西污水厂）
- 产品：3 条（高压柜、低压柜、箱变）
- 人员：2 条

## 📝 使用示例

### 示例1：生成投标文件

1. 启动系统
2. 上传招标文件
3. 查看解析结果
4. 查看匹配的资料
5. 生成投标文件
6. 下载修改

### 示例2：添加案例

1. 进入"资料管理" → "案例管理"
2. 填写项目信息
3. 点击"添加案例"
4. 案例已保存到数据库
