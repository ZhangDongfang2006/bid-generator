## 🔔 自动提醒机制

系统会在以下情况提醒你：
1. **首次检查**：第一次运行检查时提示开始收集资料
2. **定期提醒**：距离上次提醒超过2天，自动提醒
3. **进度报告**：每次运行都显示当前资料收集进度

### 如何使用检查工具

#### 手动检查
```bash
cd ~/.openclaw/workspace
python3 check_materials.py
```

#### 查看详细资料需求
```bash
cat ~/workspace/bid-generator/资料需求清单.md
```

#### 查看当前进度
```bash
cat ~/.openclaw/workspace/bid-generator/data/参考资料/资料收集进度.txt
```
