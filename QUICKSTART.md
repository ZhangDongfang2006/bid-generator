# 快速启动指南

> 快速启动投标文件自动生成系统

## 方法1：使用快速启动脚本（推荐）

### 本地启动（单机使用）

```bash
# 进入项目目录
cd ~/workspace/bid-generator

# 运行启动脚本
./setup.sh
```

脚本会自动完成以下操作：
1. 检查 Python 环境
2. 创建虚拟环境
3. 安装依赖包
4. 初始化数据库（添加测试数据）
5. 启动 Web 界面

### 局域网启动（多人共用）

如果需要让同一局域网内的其他同事访问系统：

```bash
# 进入项目目录
cd ~/workspace/bid-generator

# 运行启动脚本（局域网模式）
./setup.sh
```

启动后使用以下命令启动：

```bash
streamlit run app.py --server.address=0.0.0.0
```

其他同事可以通过浏览器访问：`http://你的IP地址:8501`

> 💡 提示：查看你的IP地址
> - Windows: `ipconfig` (查看 IPv4 地址)
> - macOS/Linux: `ifconfig | grep "inet "`

## 方法2：手动启动

### 步骤1：安装依赖

```bash
cd ~/workspace/bid-generator
pip3 install -r requirements.txt
```

### 步骤2：初始化数据库

```bash
python3 database.py
```

### 步骤3：启动应用

**本地启动：**
```bash
streamlit run app.py
```

**局域网启动：**
```bash
streamlit run app.py --server.address=0.0.0.0
```

## 常见问题

### Q: 启动失败怎么办？

A: 检查端口 8501 是否被占用，可以尝试指定其他端口：

```bash
streamlit run app.py --server.port 8502
```

### Q: 其他同事无法访问？

A: 确保以下几点：
1. 使用 `--server.address=0.0.0.0` 启动
2. 检查防火墙设置
3. 确认在同一局域网
4. 确认IP地址正确

**防火墙设置：**

**Windows:**
```
控制面板 → Windows Defender 防火墙 → 高级设置 → 入站规则 → 新建规则
选择"端口" → TCP → 8501 → 允许连接
```

**macOS:**
```
系统偏好设置 → 安全性与隐私 → 防火墙 → 防火墙选项
允许 Python 入站连接
```

**Linux (Ubuntu):**
```bash
sudo ufw allow 8501/tcp
```

A: 生成的投标文件保存在 `output/` 目录下，也可以直接在界面中下载。
