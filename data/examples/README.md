# 示例数据目录

此目录包含用于演示的示例数据，不是真实的公司数据。

## 数据说明

### 资质文件 (qualifications.json)
- 包含 2 个示例资质证书
- 示例：质量管理体系认证、环境管理体系认证

### 案例文件 (cases.json)
- 包含 2 个示例项目案例
- 示例：高压开关柜项目、预制舱项目

### 产品文件 (products.json)
- 包含 3 个示例产品
- 示例：高压开关柜、低压开关柜、箱式变电站

### 人员文件 (personnel.json)
- 包含 2 个示例人员
- 示例：技术总监、项目经理

## 注意

这些数据**仅用于演示**，**不是真实数据**。
真实数据位于 `/data` 目录（已被 .gitignore 忽略）。
