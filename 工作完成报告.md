## 如何使用系统

### 启动系统

#### 单人使用（本地）
```bash
cd ~/workspace/bid-generator
./setup.sh
```

#### 多人共用（局域网）
```bash
cd ~/workspace/bid-generator
./start_lan.sh
```

启动后访问：`http://localhost:8501`

### 查看数据统计
```bash
cd ~/workspace/bid-generator
python3 stats.py
```

2. 查看"📊 资料管理"了解已有数据

## 其他同事如何使用

### 方案1：每人独立安装（推荐）

**优点**：互不干扰，可以同时工作
**缺点**：需要每台电脑都安装，数据需要手动同步

**操作步骤**：
1. 获取项目文件包
2. 解压到本地目录
3. 运行 `./setup.sh` 安装
4. 日常使用运行 `./setup.sh` 启动
5. 定期同步 `data/` 目录数据

**数据同步**：
```bash
# 导出数据（项目负责人）
tar -czf data_backup.tar.gz data/

# 导入数据（其他同事）
tar -xzf data_backup.tar.gz
```

### 方案2：多人共用一台服务器

**优点**：数据统一，无需重复安装
**缺点**：服务器电脑需保持运行

**操作步骤**：
1. 在服务器电脑上运行 `./start_lan.sh`
2. 记录显示的IP地址（如 192.168.1.100）
3. 其他同事在浏览器访问：`http://192.168.1.100:8501`

**注意事项**：
- 确保防火墙允许8501端口
- 确保所有设备在同一局域网
- 定期备份 `data/` 目录
